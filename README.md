# Barbacoa Tatemada El Vale 🌮

Landing page moderna y responsiva para el restaurante "Barbacoa Tatemada El Vale" en Nogales, Sonora.

## Características
- **Diseño Premium**: Estética basada en tonos tierra y barro, alineada con la tradición tatemada.
- **Totalmente Responsiva**: Optimizada para dispositivos móviles con un menú interactivo estilo "dock".
- **Footer Moderno**: Estructura de 4 columnas con diseño de cintas decorativas.
- **Funcionalidades**: 
  - Mapa interactivo de Google Maps.
  - Botón de pedido directo por WhatsApp.
  - Galería de platillos con efectos de luz (*spotlight*).
  - Horarios actualizados dinámicamente.

## Tecnologías utilizadas
- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla)
- Lucide Icons (SVG)
- Google Fonts (DM Serif Display & Inter)

## Instalación
Solo clona el repositorio y abre `index.html` en tu navegador o usa una extensión como Live Server.

```bash
git clone https://github.com/tu-usuario/barbacoa-tatemada.git
```
