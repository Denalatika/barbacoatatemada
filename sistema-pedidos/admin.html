📊 Conexión Gratuita a Google Sheets (CRM)

Configura tu base de datos de clientes y pedidos en línea. Cuando un cliente ordene en el sitio, sus datos (Nombre, Teléfono, Tipo de Entrega, Dirección, Método de Pago, Pedido y Notas) **se escribirán automáticamente en una hoja de Google Excel en tu cuenta de Drive**. ¡Totalmente gratis y sin costos fijos!

🟢 ¡Integración activa! Los pedidos de tus clientes se enviarán de fondo a tu hoja de Google Sheets.

📘 Guía de Instalación Paso a Paso (Solo toma 3 minutos)

1 Crear la hoja de cálculo

Crea una hoja de cálculo nueva en tu Google Drive con el nombre "Pedidos Barbacoa". Cambia el nombre de la primera pestaña a "Pedidos".

2 Abrir el editor de Scripts

En la barra superior de tu hoja de cálculo, haz click en Extensiones y luego en Apps Script. Esto abrirá una pestaña de desarrollo de Google.

3 Copiar y Pegar el Código

Borra todo el código que aparezca en el archivo Código.gs de Apps Script y pega el siguiente bloque completo de código:

/*
 * Google Apps Script - Registro de Pedidos Barbacoa Tatemada
 * Permite guardar de forma segura pedidos hechos desde el sitio estático
 */

function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName("Pedidos");
  
  // Si no existe la pestaña, crearla
  if (!sheet) {
    sheet = SpreadsheetApp.getActiveSpreadsheet().insertSheet("Pedidos");
  }
  
  // Si la pestaña está vacía (fila 1 está vacía), crear los encabezados
  if (sheet.getLastRow() === 0) {
    sheet.appendRow([
      "Fecha/Hora", 
      "Nombre Cliente", 
      "Teléfono", 
      "Fecha Nacimiento", 
      "Tipo de Entrega", 
      "Dirección", 
      "Ubicación / Link", 
      "Método de Pago", 
      "Estado de Pago", 
      "Monto Total", 
      "Pedido", 
      "Notas"
    ]);
    sheet.getRange("A1:L1").setFontWeight("bold").setBackground("#FFF4E0").setFontColor("#B85C2C");
  }
  
  try {
    var data = JSON.parse(e.postData.contents);
    
    // Añadir fila con la información del pedido
    sheet.appendRow([
      new Date().toLocaleString("es-MX", {timeZone: "America/Hermosillo"}),
      data.customer_name || "N/A",
      "'" + (data.customer_phone || "N/A"), // La comilla simple evita que Excel borre los ceros iniciales
      data.customer_dob || "N/A",
      data.delivery_type || "N/A",
      data.address || "N/A",
      data.location_link || "N/A",
      data.payment_method || "N/A",
      data.transfer_status || "N/A",
      data.order_total || "N/A",
      data.order_items || "N/A",
      data.notes || "N/A"
    ]);
    
    // Auto-ajustar columnas
    sheet.autoResizeColumns(1, 12);
    
    return ContentService.createTextOutput(JSON.stringify({
      "status": "success",
      "message": "Pedido registrado correctamente en Google Sheets."
    })).setMimeType(ContentService.MimeType.JSON);
    
  } catch(error) {
    return ContentService.createTextOutput(JSON.stringify({
      "status": "error",
      "message": error.toString()
    })).setMimeType(ContentService.MimeType.JSON);
  }
}

// Habilitar CORS / GET para pruebas
function doGet(e) {
  return ContentService.createTextOutput(JSON.stringify({
    "status": "online",
    "message": "El Apps Script de Barbacoa Tatemada El Vale está funcionando correctamente."
  })).setMimeType(ContentService.MimeType.JSON);
}

4 Implementar como Aplicación Web

1. Haz click en el botón Implementar (Deploy) en la esquina superior derecha, luego selecciona Nueva implementación (New Deployment).
2. Haz click en el engranaje de configuración y elige Aplicación Web (Web App).
3. En el campo "Ejecutar como" selecciona: Tú (tu correo electrónico).
4. En el campo "Quién tiene acceso" selecciona obligatoriamente: Cualquiera (Anyone).
5. Presiona Implementar. Google te pedirá autorizar permisos (haz click en "Configuración Avanzada" e "Ir a Pedidos Barbacoa (no seguro)").

5 Guardar la URL en tu sitio

Al finalizar la implementación, Google te dará una URL de la Aplicación Web. Cópiala por completo, regresa a esta pestaña y pégala en el campo gris de arriba de esta página. ¡Haz click en "Guardar" y listo!
